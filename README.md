# 📝 Task Manager

A simple and clean to-do list web app to add, manage, and track daily tasks built using HTML, CSS, and JavaScript.

## 🔗 Live Demo
[View Live Project](https://f250594-stack.github.io/Task-manager-app/)

## ✨ Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks by All / Active / Completed
- Task counter showing remaining tasks
- Clear all completed tasks at once
- Data saved using LocalStorage (tasks remain after refreshing the page)

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla JS, DOM Manipulation)
